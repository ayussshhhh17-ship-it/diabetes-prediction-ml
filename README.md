# 🩺 Diabetes Prediction — Machine Learning Web App

An AI-powered diabetes risk predictor built with **Support Vector Machine (SVM)** trained on the **PIMA Indians Diabetes Dataset**. Includes a beautiful, interactive Streamlit web app for real-time predictions.

## 📋 Project Overview

This project predicts whether a person is **diabetic or non-diabetic** based on 8 clinical features using a Support Vector Machine (SVM) classifier with a linear kernel.

| Attribute | Detail |
|---|---|
| **Dataset** | PIMA Indians Diabetes Dataset |
| **Algorithm** | SVM — Linear Kernel |
| **Features** | 8 clinical inputs |
| **Training Accuracy** | ~73% |
| **Test Accuracy** | ~74% |

## 🗂️ Project Structure

```
diabetes/
├── app.py                    # Streamlit web application
├── diabetes_prediction.py    # Model training script
├── diabetes.csv              # PIMA Indians dataset
├── Diabetes.ipynb            # Original Colab notebook
├── requirements.txt          # Python dependencies
├── diabetes_model.sav        # Trained SVM model (generated)
└── scaler.sav                # Fitted StandardScaler (generated)
```

> **Note:** `diabetes_model.sav` and `scaler.sav` are generated by running `diabetes_prediction.py`. They are excluded from version control.

## 🔬 Input Features

| Feature | Description | Unit |
|---|---|---|
| Pregnancies | Number of times pregnant | — |
| Glucose | Plasma glucose concentration | mg/dL |
| BloodPressure | Diastolic blood pressure | mm Hg |
| SkinThickness | Triceps skin fold thickness | mm |
| Insulin | 2-Hour serum insulin | mu U/ml |
| BMI | Body Mass Index | kg/m² |
| DiabetesPedigreeFunction | Genetic risk score | — |
| Age | Age of the patient | years |

**Output:** `0` → Non-Diabetic &nbsp; | &nbsp; `1` → Diabetic

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/<your-username>/diabetes-prediction.git
cd diabetes-prediction
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Train the model
```bash
python diabetes_prediction.py
```
This generates `diabetes_model.sav` and `scaler.sav`.

### 4. Launch the web app
```bash
streamlit run app.py
```
Open **http://localhost:8501** in your browser.

## 🤖 Model Pipeline

```
Raw Data (diabetes.csv)
       ↓
StandardScaler (fit on training data)
       ↓
SVM Classifier (kernel='linear')
       ↓
Train/Test Split (80/20, stratified)
       ↓
Evaluation (accuracy_score)
       ↓
Pickle Export (diabetes_model.sav + scaler.sav)
```
